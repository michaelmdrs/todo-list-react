# Todo-List

## Descrição
O Todo-List é um projeto simples, com funcionalidades básicas. 
O objetivo era implementar os conceitos de React para reforça o conceito e práticas aprendidos.

## Funcionalidades
A funçao principal da aplicação é adicionar tarefas do dia-a-dia, no momento está cumprindo o que foi projetado, mas já estou
estudando outras formas de adicionar outras funcionalidades, como disparar notificações, integrar o Google Calendar para gerenciar tarefas em datas especificas.

## Tecnologias
- JavaScript
- React
- TailwindCSS
- Vite
